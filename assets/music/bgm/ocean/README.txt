海洋地图背景音乐目录
======================

将您的海洋主题背景音乐文件放在此目录下。

支持的文件格式：
- MP3 (推荐)
- WAV
- OGG
- FLAC

文件命名规则：
------------
1. 将音乐文件命名为：ocean.mp3 或 ocean.wav 或 ocean.ogg 或 ocean.flac
2. 程序会自动识别并加载此文件作为海洋地图的背景音乐

示例：
- ocean.mp3
- ocean.wav

如果此目录下没有音乐文件，游戏将使用默认的背景音乐（background.wav）。

替换方法：
----------
1. 将您自己的音频文件复制到此目录
2. 确保文件名与上述一致
3. 运行游戏即可听到新的海洋地图背景音乐
